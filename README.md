## Objectives

본 강좌에서는 경제학과 통계학 교수가 ‘데이터와 사회과학’이라는 주제에 대해 공동으로 강의한다. 사회 전반의 다양한 사안을 토론하고 그에 관련된 자료를 얻고 분석하는 방법을 배움으로써 사회를 보다 깊이 있게 바라보는 통찰력을 기르고 데이터 리터러시(data literacy)를 함양하는 것을 목표로 한다.
